Intro

Hello! Thanks for taking the time out to get to know me. I am Sonali, a Software Engineer with a passion in Data Analytics. I am working as a Data Scientist Intern in the semiconductor industry while pursuing an MS in Data Analytics at San José State University. Before shifting to the USA to pursue higher studies, I worked as a multidisciplinary technology associate in a startup back in India working on multiple projects while mentoring interns.

Projects

Social Media Popularity Prediction - Solution ranked #1 on leaderboard of SMPD challenge

The Social Media Popularity Prediction task focuses on predicting the impact of sharing different posts for a publisher on social media. Given a photo (a.k.a. post) from a publisher, the goal is to automatically predict the popularity of the photo, e.g., view count for Flickr, Pin count for Pinterest, etc. Visit Project.

Online Vs Offline Learning - Sentiment Analysis

To analyze the overall opinion of the student community with respect to online learning introduced during the pandemic as compared to traditional offline learning, fetched data from twitter, historical data from previous surveys and collected data from San José State University students was used. Visit Project.

Fradulent Account Detection - Ethereum Blockchain

This project aims to identify fraudulent accounts in the Ethereum blockchain using a combination of data from Kaggle and data scraped using the Etherscan API. Visit Project.

Air Quality Index Prediction

This project uses data fetched from AirNow API in the state of California to predict Air Quality Index of any given region in the state, by using data at the granularity level of latitude and longitude. Visit Project.

Trend Analysis of Ukraine-Russia War

This project collates data related to the war from various true sources in one repository for analyzing trends. All data are official and additionally modeled by us. Visit Project.
